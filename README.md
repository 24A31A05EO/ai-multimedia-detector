# AI Multimedia Authenticity Detection System

## Overview
This project detects AI-generated multimedia content including images, videos, audio, text, emails, and URLs. It is built using **FastAPI** (backend), **React** (frontend), and **Google Gemini Pro** for AI analysis. The project is beginner-friendly, cloud-ready, and does **not require any paid services**.

## Features
- Detect AI-generated images and videos
- Detect synthetic audio
- Analyze text, emails, and URLs for AI patterns
- User-friendly web interface with upload and result display
- GitHub-based collaborative workflow

## Tech Stack
- **Backend:** FastAPI, Python, TensorFlow, MediaPipe
- **Frontend:** React, Axios
- **AI Tools:** Google Gemini Pro
- **Database/Auth:** Firebase (Free Tier)
- **Version Control:** Git & GitHub
